Explain what pointers are in C.

A pointer is a variable that stores the memory address of another variable.

Instead of holding a direct value like an int or char, a pointer holds the location in memory where that value is stored.

In C, pointers are declared using the * symbol. For example, int *p; means that p is a pointer to an integer. To assign a pointer, you usually use the address-of operator &, which gives the memory address of a variable. For instance, if you have int x = 10;, then p = &x; makes p point to x.

To access or modify the value stored at the memory address a pointer points to, you use dereferencing, also done with *. Writing *p gives you the value of x, and changing *p = 20; will update x itself.
